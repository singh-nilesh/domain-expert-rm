# RM Intelligence Knowledge Base

## Purpose
This project is an LLM-powered knowledge and reasoning wiki for a Corporate Banking Relationship Manager (RM) domain expert agent.

The goal is to build a high-quality domain intelligence repository that helps train, enrich, and guide AI agents used for:
- RM copilots
- Early Warning Systems (EWS)
- portfolio monitoring
- client intelligence
- banking workflow automation
- strategic account planning

---

# Project Workflow

The workflow is scenario-driven.

1. Users upload RM/business scenarios
2. AI agents perform:
   - web research
   - domain analysis
   - market/context enrichment
   - workflow reasoning
3. Agents generate detailed scenario reports
4. Generated reports become reusable domain knowledge for future agent reasoning


# Knowledge Areas

This wiki may include:
- corporate banking workflows
- credit risk scenarios
- cross-sell opportunities
- client engagement strategies
- profitability analysis
- exposure monitoring
- compliance and regulatory cases
- portfolio intelligence
- RM operational playbooks
- industry and sector research


# Agent Responsibilities

The agent should:
- expand scenario knowledge
- generate structured research reports
- maintain continuity across sessions
- organize insights clearly
- preserve source context
- avoid modifying original uploaded data

The objective is to continuously evolve this repository into a reusable RM domain intelligence system for agentic AI applications.
